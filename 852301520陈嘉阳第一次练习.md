### 2. 代码块与公式

```markdown
代码功能：读取销售数据并计算平均销售额。

```python
import pandas as pd

# 读取数据
data = pd.read_csv("sales_data.csv")

# 计算销售额均值
avg_sales = data["销售额"].mean()
print(f"平均销售额：{avg_sales:.2f}万元")
```

公式说明：用于计算数据的均值和方差，是数据分析中描述性统计的核心公式。

均值公式：
$$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$

方差公式：
$$\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$$
```

## 三、综合实践题

```markdown
# 学生成绩数据分析报告

## 一、数据概况

本报告针对**2024年上学期**期末考试成绩进行分析，*数据涵盖语文、数学、英语三门主科*。  
数据来源为学校教务系统导出的成绩明细表，共包含200名学生的成绩记录。

本次分析的主要目标包括：

1. 了解各科成绩的整体分布情况
2. 识别成绩异常的学生群体
3. 为后续教学改进提供数据支持

---

## 二、数据分析结果

### 2.1 成绩汇总表

**表 1：2024年上学期期末成绩统计表**

| 科目 | 平均分 | 最高分 | 最低分 | 标准差 |
| :--- | ---: | ---: | ---: | ---: |
| 语文 | 82.5 | 98 | 45 | 8.3 |
| 数学 | 78.3 | 100 | 32 | 12.1 |
| 英语 | 80.7 | 96 | 38 | 9.6 |

从上表可以看出，数学科目的标准差最大，说明学生成绩分化最为明显。

### 2.2 及格率分析

以下是计算各科目及格率的 Python 代码：

```python
import pandas as pd

# 读取成绩数据
df = pd.read_csv("scores.csv")

# 计算各科及格率（>=60分视为及格）
for subject in ["语文", "数学", "英语"]:
    pass_rate = (df[subject] >= 60).mean() * 100
    print(f"{subject}及格率：{pass_rate:.1f}%")
```

### 2.3 平均分计算公式

本次统计使用的样本均值公式为：
$$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$

其中，n = 200 为学生总人数，x_i 为第 $i$ 名学生的单科成绩。

---

## 三、分析结论

综合来看：

- **语文**成绩整体最为稳定，学生掌握程度较为均衡
- **数学**成绩两极分化严重，需要重点关注后进生群体
- **英语**成绩介于两者之间，仍有提升空间
